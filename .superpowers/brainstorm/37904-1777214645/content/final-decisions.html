3 分发渠道 ★ GitHub + Homebrew

开源 Mac 工具的标配组合 · 不走 Mac App Store

① GitHub Releases

每个 tag 自动构建 .dmg + .zip 上传 · GitHub Actions 全自动化签名（个人开发者证书 / Sparkle 自动更新）

✓ 主分发渠道

② Homebrew Cask

brew install --cask bubblekeys · 开源 Mac 软件的「正确打开方式」· 提交一个 cask formula 即可

✓ 第二渠道（必备）

③ Mac App Store

需要 99/年的开发者账号 · App Store 沙箱不允许全局键盘监听 · 本质上做不了

✗ 跳过
签名方案：不强制 Apple notarization（要 99/年）· 走「开源项目自签名 + 用户 ⌃右键打开」路线 · README 写清楚步骤
未来如果有赞助：升级到 notarized DMG（一键打开免警告）
